Organiza.

Política de Privacidade

Última atualização: maio de 2026  ·  Versão 1.0

Resumo: A Organiza coleta apenas os dados necessários para que você gerencie seus leads no WhatsApp Web. Seus dados nunca são usados para publicidade, nunca são vendidos e nunca são compartilhados com terceiros para fins comerciais.

1. Quem somos

A Organiza é uma extensão para Google Chrome desenvolvida por Pedro Pires de La Corte (pedropiresdelacorte03@gmail.com), com sede no Brasil. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações ao utilizar a extensão Organiza.

Para fins da Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), o desenvolvedor é o Controlador dos seus dados pessoais. O serviço de banco de dados Supabase Inc. atua como Operador.

2. Dados coletados

CategoriaDados específicosFinalidade
Autenticação E-mail, senha (nunca armazenada em texto puro — processada pelo Supabase Auth), token JWT de sessão Identificar e autenticar o corretor na extensão e no painel web

Cache local Classificações, snooze de leads, preferências de exibição Melhorar a experiência — armazenados apenas no dispositivo via chrome.storage.local

3. Como os dados são usados

Os dados coletados são usados exclusivamente para:

  • Autenticar o usuário e manter a sessão ativa na extensão e no painel web
  • Criar, exibir, atualizar e excluir leads no banco de dados do corretor
  • Calcular e exibir métricas de desempenho de vendas (conversão, VGV, leads por período)
  • Gerar alertas de follow-up para leads sem contato recente
  • Sincronizar dados entre dispositivos do mesmo usuário autenticado

Declaração de Uso Limitado (Chrome Web Store — Limited Use Policy):
Os dados coletados pela Organiza nunca são usados para publicidade personalizada, retargeting ou interest-based advertising. Os dados nunca são vendidos, repassados ou compartilhados com terceiros para qualquer finalidade comercial ou publicitária. O uso é estritamente limitado às funcionalidades descritas nesta política.

4. Compartilhamento com terceiros

ServiçoEmpresaPaísFinalidade
Supabase Supabase Inc. Estados Unidos Banco de dados de leads, autenticação de usuários, armazenamento seguro

Os dados são transmitidos ao Supabase por conexão criptografada (HTTPS/TLS). Para mais detalhes, consulte a Política de Privacidade do Supabase.

Transferência internacional: Ao usar a Organiza, seus dados são armazenados em servidores do Supabase Inc. nos Estados Unidos. Ao aceitar esta política na tela de consentimento da extensão, você consente expressamente com essa transferência.

5. Base legal (LGPD)

  • Consentimento explícito (art. 7º, I) — coletado na tela de aceite exibida na primeira utilização da extensão. Pode ser revogado a qualquer momento solicitando a exclusão da conta.
  • Execução de contrato (art. 7º, V) — para os dados estritamente necessários para prestar o serviço contratado.

6. Segurança dos dados

  • Toda comunicação com o Supabase usa HTTPS/TLS
  • Senhas nunca são armazenadas em texto puro — gerenciadas com hash seguro pelo Supabase Auth
  • O acesso ao banco é controlado por Row Level Security (RLS) — cada usuário acessa apenas seus próprios leads
  • Tokens JWT têm validade limitada e são renovados automaticamente
  • Cache local armazenado em chrome.storage.local — inacessível a outras páginas ou extensões

7. Retenção de dados

Os dados são mantidos enquanto sua conta estiver ativa. Ao solicitar a exclusão da conta, todos os dados associados (leads, qualificações, histórico) são permanentemente excluídos do banco dentro de 30 dias.

Dados de cache no dispositivo são apagados automaticamente ao fazer logout na extensão ou ao desinstalá-la.

8. Seus direitos (LGPD)

  • Acesso: solicitar cópia de todos os dados mantidos sobre você
  • Correção: solicitar correção de dados inexatos ou desatualizados
  • Exclusão: solicitar a remoção permanente dos seus dados
  • Portabilidade: solicitar exportação dos dados em formato estruturado
  • Revogação do consentimento: revogar o consentimento dado anteriormente

Para exercer esses direitos, envie um e-mail para pedropiresdelacorte03@gmail.com com o assunto "Solicitação LGPD". Respondemos em até 15 dias úteis.

9. Crianças e adolescentes

A extensão Organiza é destinada exclusivamente a profissionais adultos. Não coletamos intencionalmente dados de menores de 18 anos. Caso isso ocorra, entre em contato para exclusão imediata.

10. Alterações nesta política

Esta política pode ser atualizada periodicamente. Alterações relevantes serão comunicadas via extensão. A data da última atualização estará sempre indicada no topo desta página.
